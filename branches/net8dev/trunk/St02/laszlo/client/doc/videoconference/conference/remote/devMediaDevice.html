<devMediaDevice>

Media device, an abstract class.

Extends node
Attributes
Name Usage Type (Tag) Type (JS) Default Category
allowed
 

Shared flag defined in mediadevice.prototype that tracks whether the user has allowed device access. Default: false. True after user has accepted security dialog, false if the user denies access with security dialog. Note: allowed is not defined in objects, but in the prototype. Initialized in the script at the end of the file. @keywords readonly

deviceindex null
 

Zero-based index of the device, or null for user's default device. Default is null, which selects the default device. When the default device is initialized, deviceindex is updated to its actual index.

devicename string
 

Name of the device. @keywords readonly

capturing true
 

Boolean flag that controls if the media device is currently capturing input.

active false
 

Flag that tracks if there's device activity (audio sound, video motion). @keywords readonly

_setDeviceIndex
devMediaDevice._setDeviceIndex

Setter for the device index attribute (0-based index, or null). If the index is null, the default device is selected. Changing deviceindex will try to open a different device, and set devicename if successful. Subclasses should implement this device specific method. @keywords private


_setCapturing
devMediaDevice._setCapturing

Setter for capturing attribute (true or false boolean). Subclasses should implement this device specific method. TODO: Move this to subclasses. @keywords private


_onStatusStatic
devMediaDevice._onStatusStatic

Static method called on mediadevice.prototype, the handler for the Flash Camera onStatus callback. Info is the Flash status event name, like "Device.Action". This notifies all mediadevice instances that the allowed status changed. Don't call this on a mediadevice instance! @keywords private


_updateAllAllowedStatic
devMediaDevice._updateAllAllowedStatic

Static method called on mediadevice.prototype, that sets the allowed class variable and tells all mediadevices that allowed changed. Don't call this on a mediadevice instance! @keywords private


_updateAllowed
devMediaDevice._updateAllowed

Called on each mediadevice instance when the shared mediadevice.prototype.allowed value changes. @keywords private
